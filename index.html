개념 정리

JavaScript의 핵심 개념들을 체계적으로 학습합니다.

  • 📖 JavaScript 기초 문법 학습
  • 📝 변수 (Variables) - var, let, const
  • 🏷️ 타입 (Types) - 원시타입 & 참조타입
  • ⚡ 연산자 기초 (Operators) - 산술, 비교, 논리, 할당
  • 🚀 연산자 중급 (Operators+) - 추가 실습
  • 🎯 연산자 고급 (Operators++) - 심화 실습
  • 🔀 조건문 (Conditionals) - if, switch, 삼항연산자
  • 🔄 반복문 (Loops) - for, while, for...of
  • 🎯 배열 메서드 (Array Methods) - forEach, map, filter, reduce
