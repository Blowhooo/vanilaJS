🔍 간단한 상품 검색 프로그램

프로그램 설명

상품 목록에서 검색어가 포함된 항목을 찾아 필터링하는 검색 시스템입니다.

주요 기능

  • 🔤 대소문자 무시: Apple, apple, APPLE 모두 동일하게 처리
  • 📍 부분 매칭: includes() 메서드로 간단하게 검색
  • 📊 통계 분석: 검색 결과 개수와 적중률 계산
  • 📈 진행도 표시: 막대 그래프로 시각화
  • 💡 검색 안내: 결과에 따른 메시지 표시

프로그램 동작 과정

  • 1️⃣ 검색어를 소문자로 변환 (toLowerCase() 사용)
  • 2️⃣ for문으로 각 상품 순회
  • 3️⃣ 각 상품명도 소문자로 변환
  • 4️⃣ includes() 메서드로 포함 여부 확인
  • 5️⃣ 검색된 상품 카운트 및 출력

핵심 코드 - 간단한 검색

// 각 상품을 하나씩 확인
for (let i = 0; i < products.length; i++) {
    let product = products[i];
    let productLower = product.toLowerCase();
    
    // includes() 메서드로 간단하게 검색
    if (productLower.includes(searchLower)) {
        foundCount = foundCount + 1;
        console.log('  ✅ ' + foundCount + '. ' + product);
    }
}

💻 실습 방법

⚠️ 알림: 프로그램은 페이지 로드시 자동 실행됩니다!

  2. Console 탭 선택: 상단 메뉴에서 Console 탭을 클릭합니다

📚 학습된 개념들

이 프로그램을 통해 배우는 것들:

  • 📌 문자열 메서드: toLowerCase()로 대소문자 통일
  • 📌 배열 순회: for문으로 배열 요소 하나씩 확인
  • 📌 문자열 검색: includes() 메서드 활용
