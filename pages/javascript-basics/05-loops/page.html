🔐 비밀번호 유효성 검사 프로그램

프로그램 설명

입력된 비밀번호가 보안 기준을 충족하는지 종합적으로 검사하는 프로그램입니다.

검사 항목 (5가지 조건)

  • 📏 길이 검사: 8자 이상인지 확인
  • 🔠 대문자 검사: A-Z 포함 여부 및 개수 확인
  • 🔡 소문자 검사: a-z 포함 여부 및 개수 확인
  • 🔢 숫자 검사: 0-9 포함 여부 및 개수 확인
  • ⚡ 특수문자 검사: !@#$%^&* 등 포함 여부 확인

프로그램 진행 과정

  • 1️⃣ for문으로 문자열 길이 직접 계산
  • 2️⃣ 각 문자를 순회하며 문자 타입 확인
  • 3️⃣ ASCII 코드 비교로 대/소문자 구분
  • 4️⃣ 특수문자 목록과 대조하여 검증
  • 5️⃣ 진행도 바 생성 및 최종 평가

핵심 코드 구조

// 1. 길이 계산 (for문 활용)
for (let i = 0; password[i] !== undefined; i++) {
    passwordLength = passwordLength + 1;
}

// 2. 대문자 검사 (ASCII 비교)
for (let i = 0; i < passwordLength; i++) {
    if (char >= 'A' && char <= 'Z') {
        hasUpperCase = true;
        upperCount = upperCount + 1;
    }
}

// 3. 특수문자 검사 (이중 반복문)
for (let i = 0; i < passwordLength; i++) {
    for (let j = 0; j < specialChars.length; j++) {
        if (char === specialChars[j]) {
            hasSpecial = true;
            break;
        }
    }
}
