배열 메서드 (Array Methods)

JavaScript의 강력한 배열 메서드를 활용하여 데이터를 효율적으로 처리하는 방법을 학습합니다.

  • forEach, map, filter, find, reduce 등 핵심 메서드 이해
  • 실무에서 자주 사용되는 데이터 처리 패턴 학습
  • 메서드 체이닝을 통한 복잡한 데이터 처리

과제 1: 학생 성적 통계 시스템

배열 메서드를 활용하여 학생 데이터를 분석하고 통계를 계산합니다.

요구사항

  • • reduce를 사용하여 평균 성적 계산
  • • filter를 사용하여 우수 학생 필터링
  • • map과 reduce를 조합하여 전공별 통계 생성

과제 2: 온라인 쇼핑몰 상품 관리

filter, map 메서드를 사용하여 동적 상품 필터링 시스템을 구현합니다.

요구사항

  • • 카테고리별 상품 필터링
  • • 가격 범위로 상품 검색
  • • 검색어로 상품명 필터링

과제 3: 장바구니 총액 계산

reduce와 find 메서드를 활용하여 장바구니 기능을 구현합니다.

요구사항

  • • find로 상품 정보 조회
  • • reduce로 총액 계산
  • • map으로 장바구니 아이템 표시

과제 4: 메서드 체이닝 실습

여러 배열 메서드를 연결하여 복잡한 데이터 처리를 수행합니다.

실습 내용

  • • filter → map → reduce 체이닝
  • • 조건부 필터링과 변환
  • • 최종 결과 집계
