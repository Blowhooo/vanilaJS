1. 회사 조직도 관리 시스템

복잡한 계층 구조 + 다중 관계 처리

중급

부서-직원-프로젝트의 3중 관계를 관리하는 시스템입니다. 재귀 함수와 트리 구조 처리를 중점적으로 다룹니다.

핵심 학습 내용

  • 재귀 함수 - 조직도 트리 구조 생성
  • for...of, forEach, map - 반복문 조합 활용
  • 깊은 복사 vs 얕은 복사 - 객체 참조 이슈 해결
  • 메모리 누수 방지 - 이벤트 리스너 정리
프로젝트 시작하기 예상 소요: 1-2주
🛒

2. 전자상거래 주문 시스템

복잡한 비즈니스 로직 + 상태 관리

고급

장바구니, 재고, 쿠폰, 배송을 연동하는 완전한 주문 시스템입니다. 복잡한 비즈니스 로직과 트랜잭션 처리를 학습합니다.

핵심 학습 내용

  • reduce - 복잡한 가격 계산 로직
  • while문 - 재고 차감 로직
  • try-catch 중첩 - 결제 트랜잭션 시뮬레이션
  • 옵셔널 체이닝 - 안전한 객체 접근
프로젝트 시작하기 예상 소요: 2-3주
📊

3. 데이터 분석 대시보드

대용량 데이터 처리 + 성능 최적화

전문가

10,000개 이상의 데이터를 실시간으로 처리하는 대시보드입니다. 성능 최적화와 메모리 관리 기법을 중점적으로 다룹니다.

핵심 학습 내용

  • filter, map, reduce - 체이닝 최적화
  • 디바운싱 - 검색 성능 향상
  • 가상 스크롤 - 메모리 사용량 관리
  • Web Worker - 메인 스레드 블로킹 방지
프로젝트 시작하기 예상 소요: 2-3주
🎮

4. 실시간 게임 스코어보드

실시간 처리 + 복잡한 상태 동기화

고급

실시간으로 업데이트되는 게임 스코어보드입니다. WebSocket 시뮬레이션과 복잡한 상태 관리를 학습합니다.

핵심 학습 내용

  • WebSocket 시뮬레이션 - setInterval 활용
  • switch문 - 게임 상태 관리
  • for...in - 플레이어 데이터 순회
  • 클로저 - 점수 계산 로직
프로젝트 시작하기 예상 소요: 2-3주
📝

5. 협업 문서 편집기

복잡한 DOM 조작 + 충돌 처리

전문가

실시간 협업이 가능한 문서 편집기입니다. 모든 고급 기법이 통합된 최종 프로젝트입니다.

핵심 학습 내용

  • contentEditable - 에디터 구현
  • Debouncing - 자동 저장
  • 충돌 감지 - 동시 편집 처리
  • 히스토리 관리 - 되돌리기/다시하기
프로젝트 시작하기 예상 소요: 3-4주
