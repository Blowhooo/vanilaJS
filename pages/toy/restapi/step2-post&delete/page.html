REST API 실습

REST API를 활용하여 데이터를 가져오고 화면에 표시해보자

  • fetch API를 사용하여 데이터 요청하기
  • JSON 데이터를 파싱하여 화면에 표시하기
  • 로딩 상태와 에러 처리하기
  • async/await를 활용한 비동기 처리

과제 1: 게시글 작성하기 (POST)

새로운 게시글을 서버에 POST 요청으로 전송하세요.

요구사항

  • • API URL: http://localhost:3001/posts
  • • POST 메서드 사용
  • • JSON 형태로 데이터 전송 (title, body, userId 포함)
  • • 응답 결과 표시
  • • 로딩 상태와 에러 처리 구현

과제 2: 게시글 삭제하기 (DELETE)

지정한 ID의 게시글을 DELETE 요청으로 삭제하세요.

요구사항

  • • API URL: http://localhost:3001/posts/{id}
  • • DELETE 메서드 사용
  • • 삭제 전 확인 메시지 표시
  • • 삭제 결과 표시
  • • 에러 처리 구현
