REST API 실습

REST API를 활용하여 데이터를 가져오고 화면에 표시해보자

  • fetch API를 사용하여 데이터 요청하기
  • JSON 데이터를 파싱하여 화면에 표시하기
  • 로딩 상태와 에러 처리하기
  • async/await를 활용한 비동기 처리

과제 1: 사용자 목록 가져오기

json-server를 사용하여 사용자 목록을 가져와 화면에 표시하세요.

요구사항

  • • API URL: http://localhost:3000/users
  • • fetch API 사용
  • • 사용자 이름과 이메일 표시
  • • 로딩 상태 표시

과제 2: 게시글 작성하기

새로운 게시글을 서버에 POST 요청으로 전송하세요.

요구사항

  • • API URL: http://localhost:3000/posts
  • • POST 메서드 사용
  • • JSON 형태로 데이터 전송
  • • 응답 결과 표시

과제 3: 에러 처리

잘못된 API 요청에 대한 에러 처리를 구현하세요.

요구사항

  • • try-catch 구문 사용
  • • 네트워크 에러 처리
  • • HTTP 상태 코드 확인
  • • 사용자에게 에러 메시지 표시
